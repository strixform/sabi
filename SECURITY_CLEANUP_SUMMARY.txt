NEXT STEPS - PRODUCTION DEPLOYMENT
================================================================================

IMMEDIATE (Required to deploy):

1. GET NEW FLUTTERWAVE LIVE KEYS
   Contact: Flutterwave Dashboard → Settings → API Keys
   Required keys:
   - FLW_SECRET_KEY
   - NEXT_PUBLIC_FLW_PUBLIC_KEY
   - FLW_WEBHOOK_HASH

2. CONFIGURE VERCEL ENVIRONMENT VARIABLES
   Vercel Dashboard → SABI Project → Settings → Environment Variables

   For Production environment, add:
   - FLW_SECRET_KEY = pk_live_...
   - NEXT_PUBLIC_FLW_PUBLIC_KEY = pk_live_...
   - FLW_WEBHOOK_HASH = webhook_secret...
   - TURSO_AUTH_TOKEN = your_token
   - RESEND_API_KEY = your_key
   - NEXT_PUBLIC_APP_URL = https://sability.io
   - NODE_ENV = production

3. CONFIGURE FLUTTERWAVE WEBHOOK
   Flutterwave Dashboard → Settings → Webhooks
   Add URL: https://sability.io/api/sabi/wallet/webhook
   Event: charge.completed

4. RUN PRODUCTION TESTS
   - User registration → email verification
   - Wallet funding → payment completion
   - Order placement → campaign creation
   - Webhook receipt → wallet credit

5. DEPLOY TO PRODUCTION
   git push origin main
   (Vercel auto-deploys)

OPTIONAL (Recommended):

- Configure monitoring alerts
- Set up automated backups
