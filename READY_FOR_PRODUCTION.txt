TROUBLESHOOTING
================================================================================

If something goes wrong:

PAYMENT FAILS:
  1. Check FLW_SECRET_KEY in Vercel is set correctly
  2. Check NEXT_PUBLIC_FLW_PUBLIC_KEY is set correctly
  3. Redeploy: git push origin main
  4. Check Flutterwave dashboard for account status

WEBHOOK NOT RECEIVED:
  1. Verify webhook URL in Flutterwave: https://sability.io/api/sabi/wallet/webhook
  2. Check FLW_WEBHOOK_HASH is set in Vercel
  3. Check Flutterwave webhook logs
  4. Check Vercel function logs

WALLET NOT CREDITED:
  1. Check webhook received in Flutterwave logs
  2. Check Vercel function logs for errors
  3. Check database transaction record
  4. Manually verify payment in Flutterwave

ROLLBACK PROCEDURE:
  Option 1: Vercel Dashboard → Deployments → [Previous] → Redeploy
  Option 2: git revert HEAD && git push origin main
