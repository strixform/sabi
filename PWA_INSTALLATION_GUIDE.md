# 📱 SABI Progressive Web App (PWA) Installation Guide

SABI is now available as a Progressive Web App! Install it directly to your home screen for the best experience.

---

## 🚀 How to Install SABI

### **On Android (Chrome, Firefox, Edge, Samsung Internet)**

1. **Open SABI in your browser**: https://sability.io
2. **Look for the install prompt** (appears at bottom of dashboard after 3 seconds)
3. **Click "Install"** button
4. **Tap "Install"** in the confirmation dialog
5. **SABI will be added to your home screen** ✅

**Alternative method:**
- Tap **⋮ (menu)** → **"Install app"** or **"Add to Home Screen"**

---

### **On iPhone/iPad (Safari)**

1. **Open SABI in Safari**: https://sability.io
2. **Tap the Share button** (bottom-right icon)
3. **Scroll down and tap "Add to Home Screen"**
4. **Tap "Add"** in the confirmation
5. **SABI will appear on your home screen** ✅

---

### **On Desktop (Windows/Mac)**

1. **Open SABI in Chrome, Edge, or Brave**: https://sability.io
2. **Look for the install icon** (address bar or menu)
   - Chrome: Click the **⬇️** icon in address bar
   - Edge: Click **⬇️ Install this site as an app**
   - Brave: Click the **+** in address bar
3. **Click "Install"**
4. **SABI will open as a desktop app** ✅

## ❓ FAQ

### **Q: Is my data safe?**
✅ Yes! PWA uses the same security as the website. All data is encrypted.

### **Q: Will my account sync across devices?**
✅ Yes! Your account syncs automatically. Install on multiple devices.

### **Q: Can I uninstall it?**
✅ Yes! Uninstall just like any app:
- **Android**: Long press app → Uninstall
- **iPhone**: Long press app → Remove App → Delete App
- **Desktop**: Right-click app in taskbar/dock → Uninstall

### **Q: Does it work offline?**
✅ Partially! You can view cached pages, but payment/orders require internet.

### **Q: Is it smaller than a real app?**
✅ Yes! PWA is ~5MB vs 100MB+ for native apps.

### **Q: Which phones/browsers support PWA?**
✅ All modern phones:
- **Android 5+**: Chrome, Firefox, Edge, Samsung Internet
- **iPhone 14+**: Safari (iOS 16.4+)
- **Desktop**: Chrome, Edge, Brave, Opera

### **Q: I don't see the install prompt**
📌 **Options:**
1. Look in browser menu for "Install app" option
2. Try a different browser (Chrome recommended)
3. Clear browser cache and refresh
4. Make sure you're on https://sability.io
